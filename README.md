# RecipePlease

## 개요
1. 레시피를 공유/판매 하는 앱을 제작한다.
2. 레시피에 사용된 굿즈 상품을 판매하는 앱을 제작한다.

## Languages & Framework(Library)
- Mobile
  - React Native(TypeScript)
  - Spring Boot
  - My SQL
- Web
  - HTML/CSS/JavaScript
  - Spring Boot
  - My SQL
- Access to Database
  - MyBatis

## 주요 기능
- 일반 로그인 / 카카오 계정으로 로그인(기타 소셜 로그인)
- 레시피 업로드 및 내부 화폐로 결제
- 굿즈를 카드결제로 구매
- 기타
